# Voting Management System (Python)

## Description
This project is a Voting Management System developed using Python and MySQL to ensure secure and accurate digital voting.

## Features
- Voter registration and login
- Secure vote casting
- One person one vote validation
- Candidate management
- Result generation

## Technologies Used
- Python
- MySQL
- SQL

## How to Run
1. Install Python
2. Configure MySQL database
3. Run the application

## Purpose
To provide a secure and transparent voting system with accurate results.
